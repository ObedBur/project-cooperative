Aide & Support

Questions fréquentes

  • Comment ajouter un membre ?
    Rendez-vous dans "Gestion des Membres" puis cliquez sur "Ajouter un membre".
  • Comment modifier les informations d'un administrateur ?
    Allez dans "Gérer les administrateurs", recherchez l'administrateur puis cliquez sur "Modifier".
  • Comment consulter les rapports mensuels ?
    Cliquez sur "Rapport Mensuel" dans le menu principal.
  • Comment changer le mot de passe ?
    Contactez un super administrateur ou le support technique.
